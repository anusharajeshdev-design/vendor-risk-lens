# Vendor-Risk Lens (VRL)

Vendor-Risk Lens (VRL) is an enterprise-style Vendor Risk & Operations Monitoring Platform built using:

- ASP.NET Core 8 Web API
- React + Vite
- SQL Server
- Entity Framework Core
- Swagger
- GitHub

## Project Goal

VRL helps organizations:

- track vendor-related operational incidents
- manage vendor risks
- monitor issue statuses
- maintain audit visibility
- generate operational insights
- streamline vendor communication workflows

## Planned Features

- Vendor Management
- Incident Management
- Dashboard & KPIs
- Email Notifications
- AI-Powered Search
- Excel Export
- Audit Timeline
- Risk Scoring
